Java嵌套类

The Java programming language allows you to define a class within another class. Such a class is called a nested class.

Like this:

分类

嵌套类被分为两类：

  1. static(called static nested classes)
  2. non-static(called inner classes)

Like this:

为什么使用嵌套类？

» 提高效率。如果一个类只对某个类有用，把这个类作为内部类，是符合逻辑的，且 makes their package more streamlined.

» 增强封装性。将A类作为B类的内部类，A类可直接访问B类的private字段，并且A类对外不可见。

» 增加代码的可读性和可维护性。代码就近原则，类更靠近使用它的地方。

Static Nested Classes

与类方法一样，静态嵌套类不能直接使用实例变量和实例方法，而是通过对象引用使用它们。

静态嵌套类与外部类（也包括所有其他类）的实例成员交互，就像其他所有普通类与这些实例成员交互一样，可以把静态嵌套类看做普通类。事实上，静态嵌套类行为上与普通类一样，为了封装便利，才将它作为嵌套类。
